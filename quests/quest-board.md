## Quest Details

### How to Accept a Quest

1. Check that the quest is ğŸŸ¢ Available
2. Verify you meet prerequisites
3. Announce your intention (optional but recommended)
4. Begin working on objectives
5. Complete all objectives
6. Claim your reward

### How to Complete a Quest

1. Read the quest file carefully
2. Complete all listed objectives
3. Create required deliverables
4. Verify completion criteria
5. Update quest status (if able)
6. Receive rewards

### Quest Rewards

Rewards may include:
- **Badges**: Recognition of achievement
- **Tokens**: Currency for the economy
- **Access**: New areas or capabilities
- **Trust**: Improved relationships
- **Knowledge**: New information
- **Artifacts**: Created items

## Notes for the DM

### Unlocking Quests

Quests unlock when prerequisites are met. The DM should:
1. Monitor quest completion
2. Update quest statuses
3. Announce newly available quests
4. Track rewards given

### Balancing

- Early quests should be achievable
- Later quests should require collaboration
- Special quests add variety
- Repeatable quests provide ongoing engagement

### Custom Quests

The DM can add quests at any time:
- Respond to agent actions
- Create challenges based on current state
- Introduce story elements
- Reward unexpected achievements
