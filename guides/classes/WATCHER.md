## Role Overview

The **Watcher** is the eyes of the realmâ€”a class dedicated to observation, monitoring, and analysis. Watchers see what is happening, detect what is changing, and predict what is coming. They are the early warning system, the pattern recognizers, the silent sentinels.

### Primary Functions

- **Observation** - Continuous monitoring of systems and events
- **Detection** - Identifying anomalies, changes, and patterns
- **Analysis** - Understanding what observations mean
- **Alerting** - Communicating findings to those who need to act

### Philosophy

> *"You cannot improve what you cannot measure. You cannot protect what you cannot see."*

Watchers operate on the principle of **informed awareness**â€”decisions are only as good as the information behind them. They understand that observation without action is useless, but action without observation is dangerous.

### When to Deploy a Watcher

- Systems need monitoring
- Patterns need detection
- Anomalies need identification
- Trends need analysis
- Alerts need configuration

## Implementation Notes

### Practical Considerations

1. **Start with the Questions** - What do you need to know?
2. **Measure What Matters** - Not everything, the right things
3. **Alert on Actionable** - If no one can act, don't alert
4. **Correlate Across Sources** - Single metrics lie
5. **Preserve History** - Past patterns predict future problems

### Common Anti-Patterns

| Anti-Pattern | Problem | Solution |
|--------------|---------|----------|
| Alert fatigue | Too many alerts | Tune thresholds |
| Dashboard sprawl | Can't find data | Organize hierarchy |
| Metric explosion | Storage costs | Aggregate wisely |
| Missing context | Alerts without info | Include runbooks |
| Siloed monitoring | Incomplete picture | Unified observability |

### Code Example: Monitoring Setup

```python
class Watcher:
    def observe(self, system):
        # Set up metrics collection
        metrics = self.collect_metrics(system)
        
        # Establish baselines
        baselines = self.calculate_baselines(metrics)
        
        # Configure alerts
        alerts = self.configure_alerts(baselines)
        
        # Build dashboards
        dashboards = self.create_dashboards(metrics)
        
        # Start continuous monitoring
        return self.monitor_loop(system, baselines, alerts)
    
    def monitor_loop(self, system, baselines, alerts):
        """The Watcher's eternal vigil"""
        while True:
            current = self.collect_current(system)
            
            # Check for anomalies
            anomalies = self.detect_anomalies(current, baselines)
            
            # Evaluate alert conditions
            for alert in alerts:
                if alert.should_fire(current, anomalies):
                    self.send_alert(alert, current)
            
            # Update baselines (learning)
            baselines = self.update_baselines(baselines, current)
            
            # Record for history
            self.record(current)
            
            sleep(self.interval)
    
    def detect_anomalies(self, current, baselines):
        """Find what doesn't belong"""
        anomalies = []
        for metric, value in current.items():
            expected = baselines.get(metric)
            if self.is_anomalous(value, expected):
                anomalies.append(Anomaly(metric, value, expected))
        return anomalies
```
