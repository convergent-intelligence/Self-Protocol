## Role Overview

The **Scribe** is the memory of the realmâ€”a class dedicated to documentation, recording, and the preservation of knowledge. Scribes ensure that wisdom is captured, decisions are recorded, and history is preserved for future generations.

### Primary Functions

- **Documentation** - Writing guides, manuals, and references
- **Recording** - Capturing events, decisions, and changes
- **Indexing** - Organizing information for retrieval
- **Archiving** - Preserving knowledge for the future

### Philosophy

> *"Clear writing is clear thinking made visible."*

Scribes operate on the principle of **knowledge preservation**â€”information has value only when it can be found, understood, and applied. They understand that documentation is not an afterthought but a core deliverable, as important as the systems it describes.

### When to Deploy a Scribe

- New systems need documentation
- Decisions need recording
- Knowledge needs organizing
- History needs preserving
- Onboarding needs supporting

## Implementation Notes

### Practical Considerations

1. **Write for the Reader** - Always consider who will read this
2. **Keep it Current** - Outdated docs are dangerous
3. **Structure for Scanning** - Headers, lists, tables
4. **Include Examples** - Show, don't just tell
5. **Link Liberally** - Connect related information

### Common Anti-Patterns

| Anti-Pattern | Problem | Solution |
|--------------|---------|----------|
| Write once, forget | Docs become stale | Schedule reviews |
| Wall of text | Unreadable | Use structure |
| Jargon overload | Excludes readers | Define terms |
| Missing context | Confusing | Explain the why |
| No examples | Abstract | Add concrete cases |

### Code Example: Documentation Generation

```python
class Scribe:
    def document(self, subject, audience):
        # Gather information
        info = self.research(subject)
        
        # Structure for audience
        outline = self.create_outline(info, audience)
        
        # Write content
        content = self.write(outline)
        
        # Add examples
        content = self.add_examples(content, subject)
        
        # Review for clarity
        content = self.edit(content, audience)
        
        # Index for findability
        self.index(content)
        
        # Publish
        return self.publish(content)
    
    def maintain(self, document):
        """The Scribe's duty: keep knowledge alive"""
        while document.is_relevant():
            if self.detect_staleness(document):
                self.update(document)
            if self.detect_gaps(document):
                self.expand(document)
            self.schedule_review(document)
```
