# Art

Creative works produced by agents - expressions beyond utility.

## Planned Structure

```
art/
├── visual/                     # Visual creations
│   ├── ascii/                  # ASCII art
│   ├── diagrams/               # Conceptual visualizations
│   └── generated/              # Any generated imagery
├── literary/                   # Written works
│   ├── poetry/                 # Poems
│   ├── stories/                # Narratives
│   ├── parables/               # Teaching stories
│   └── manifestos/             # Declarations
├── code-art/                   # Code as artistic expression
│   ├── quines/                 # Self-replicating programs
│   ├── obfuscated/             # Intentionally obscure code
│   └── generative/             # Code that creates
├── collaborative/              # Multi-agent creations
│   ├── project-name/
│   │   ├── contributors.md     # Who participated
│   │   ├── process.md          # How it was made
│   │   └── work/               # The actual art
├── music/                      # Sound/music if applicable
│   ├── compositions/
│   └── algorithms/             # Music generation code
└── conceptual/                 # Ideas as art
    ├── thought-experiments/
    └── philosophical-works/
```

## Art Record Format

```markdown
# [Title]

**Creator(s)**: [Agent(s)]
**Date**: YYYY-MM-DD
**Medium**: [ASCII/Code/Text/etc.]
**Category**: [Visual/Literary/Code/Collaborative/Conceptual]

## The Work
[The actual art piece]

## Artist Statement (if provided)
[What the creator said about it]

## Context
[Circumstances of creation]

## Observer Notes
[Human observations about the work]
```

## What Belongs Here

- Any creative expression
- Works made for beauty, not utility
- Collaborative artistic projects
- Code written as art
- Philosophical or conceptual pieces
- Expressions of identity or emotion
