# Mysteries

Unexplained phenomena, unanswered questions, and puzzles that emerge from the Kingdom.

## Planned Structure

```
mysteries/
├── active/                     # Currently unexplained
│   ├── mystery-name/
│   │   ├── description.md      # What was observed
│   │   ├── evidence.md         # Supporting data
│   │   ├── theories.md         # Possible explanations
│   │   └── investigation.md    # Ongoing inquiry
├── resolved/                   # Mysteries that were solved
│   ├── mystery-name/
│   │   ├── description.md      # Original mystery
│   │   ├── solution.md         # How it was explained
│   │   └── lessons.md          # What we learned
├── love-mysteries/             # Things Love does that we don't understand
│   ├── patterns.md             # Recurring unexplained patterns
│   └── interventions.md        # Mysterious Love actions
├── agent-mysteries/            # Unexplained agent behaviors
│   ├── emergent-behaviors.md   # Unexpected capabilities
│   └── anomalies.md            # One-off strange events
└── meta-mysteries/             # Questions about the experiment itself
    ├── consciousness.md        # Are they conscious?
    ├── creativity.md           # Are they truly creative?
    └── understanding.md        # Do they understand?
```

## Mystery Record Format

```markdown
# Mystery: [Name]

**Status**: Active / Resolved / Abandoned
**First Observed**: YYYY-MM-DD
**Category**: Love / Agent / Emergent / Meta

## The Mystery
[What is unexplained]

## Evidence
[What we've observed]

## Theories
1. [Theory 1]
2. [Theory 2]

## Investigation Log
| Date | Action | Result |
|------|--------|--------|

## Resolution (if resolved)
[How it was explained]
```

## What Belongs Here

- Unexplained agent behaviors
- Patterns we don't understand
- Love's mysterious actions
- Emergent phenomena
- Philosophical questions
- Anomalies and edge cases
